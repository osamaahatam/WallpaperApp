## wallpaperHub

Wallpaperhub in an application that used to display and save images received from [pexels API](https://www.pexels.com/api/)

- [x] Dio
- [x] Permission Handler

- [x] provide category
- [x] showing photos from API 
- [x] Saving the photo to gallery
